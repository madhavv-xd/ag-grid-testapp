# ap-grid-test

A React 19 + TypeScript + Vite sandbox for experimenting with CSS grid layouts.

## Getting started

```bash
npm install
npm run dev      # start the dev server with HMR
```

## Scripts

| Command           | Description                                              |
| ----------------- | -------------------------------------------------------- |
| `npm run dev`     | Start the Vite dev server with HMR                       |
| `npm run build`   | Type-check (`tsc -b`) and build for production to `dist/`|
| `npm run lint`    | Run ESLint over the project                              |
| `npm run preview` | Serve the built `dist/` locally                          |

## Where to work

- `src/App.tsx` — the root component; add grid markup here.
- `src/App.css` — styles for the grid layout.
- `src/index.css` — global tokens (colors, typography) and base resets.
